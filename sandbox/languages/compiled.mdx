## Java (OpenJDK 17)

```bash
curl -X POST http://localhost:8002/api/v1/execute \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "code": "public class Main { public static void main(String[] args) { System.out.println(\"Hello\"); } }",
    "language": "java"
  }'
```

## Go (1.21)

curl -X POST http://localhost:8002/api/v1/execute \
