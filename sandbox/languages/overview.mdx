## Overview

Nadoo Sandbox supports 12+ programming languages with isolated Docker containers for each.

| Language | Version | Runtime |
|----------|---------|---------|
| Python | 3.11 | CPython |
| JavaScript | Node 20 | Node.js |
| TypeScript | Latest | ts-node |
| Java | 17 | OpenJDK |
| Go | 1.21 | go run |
| Rust | 1.74 | rustc |
| C++ | GCC 13 | g++ |
| C# | .NET 8.0 | dotnet |
| Ruby | 3.2 | ruby |
| PHP | 8.2 | php |
| SQL | PostgreSQL 16 | psql |
| Bash | 5.2 | bash |
