## API Conventions

### Import Structure

All public APIs are exposed through the main module:

```python
from nadoo_flow import (
    # Core classes
    ChainableNode,
    WorkflowExecutor,
    NodeResult,
    WorkflowContext,
    NodeContext,

    # Helper nodes
    FunctionNode,
    PassthroughNode,

    # Advanced features
    ParallelNode,
    RetryableNode,
    CachedNode,

    # And more...
)
```

### Async-First Design

All execution methods are async by default:

```python
# Async execution
result = await node.execute(node_context, workflow_context)
result = await workflow.run(input_data)

# Streaming
async for chunk in workflow.stream(input_data):
    process(chunk)
```

### Type Safety

All APIs use type hints and Pydantic models:

```python
from typing import Dict, Any, Optional
from pydantic import BaseModel

class NodeConfig(BaseModel):
    timeout: float = 30.0
    retries: int = 3
    metadata: Optional[Dict[str, Any]] = None
```

### Error Handling

Consistent error handling across all APIs:

```python
from nadoo_flow import NodeResult

try:
    # Your logic
    return NodeResult(success=True, output=data)
except Exception as e:
    return NodeResult(
        success=False,
        error=str(e),
        output={}
    )
```

## Common Patterns

### Node Definition

```python
from nadoo_flow import ChainableNode, NodeResult

class MyNode(ChainableNode):
    def __init__(self):
        super().__init__(
            node_id="my_node",
            node_type="custom",
            name="My Custom Node"
        )

    async def execute(self, node_context, workflow_context):
        # Your logic here
        return NodeResult(
            success=True,
            output={"result": "data"}
        )
```

### Workflow Composition

```python
# Using pipe operator
workflow = node1 | node2 | node3

# Using WorkflowExecutor
executor = WorkflowExecutor()
executor.add_node(node1)
executor.add_node(node2)
executor.add_edge("node1", "node2")
```

### Context Management

```python
# Workflow context for global state
context = WorkflowContext()
context.set_global_variable("user_id", "123")

# Node context for local state
node_context.input_data  # Input from previous node
node_context.output_data  # Output to next node
```

## Quick Reference

### Essential Classes

| Class | Purpose | Module |
|-------|---------|--------|
| `ChainableNode` | Base class for custom nodes | `base` |
| `WorkflowExecutor` | Workflow execution engine | `base` |
| `NodeResult` | Node execution result | `base` |
| `WorkflowContext` | Global workflow state | `base` |
| `FunctionNode` | Wrap functions as nodes | `chain` |
| `ParallelNode` | Parallel execution | `parallel` |

### Key Methods

| Method | Description | Returns |
|--------|-------------|---------|
| `node.run(input)` | Execute single node | `Dict[str, Any]` |
| `node.stream(input)` | Stream node outputs | `AsyncIterator[Dict]` |
| `workflow.execute(context)` | Execute workflow | `WorkflowContext` |
| `chain.pipe(other)` | Chain nodes with `\|` | `NodeChain` |
