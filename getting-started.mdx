## Developer Path

<Steps>
  <Step title="Set Up Development Environment">
    Install Python 3.11+ and set up your preferred IDE:

    ```bash
    # Create virtual environment
    python -m venv nadoo-env
    source nadoo-env/bin/activate  # On Windows: nadoo-env\Scripts\activate

    # Install Flow Core
    pip install nadoo-flow-core
    ```
  </Step>
  <Step title="Hello World Workflow">
    Create your first workflow in under 5 minutes:

    ```python
    from nadoo_flow import ChainableNode, NodeResult
    import asyncio

    class HelloNode(ChainableNode):
        async def execute(self, node_context, workflow_context):
            name = node_context.input_data.get("name", "World")
            return NodeResult(
                success=True,
                output={"message": f"Hello, {name}!"}
            )

    async def main():
        node = HelloNode(node_id="hello", node_type="greeting", name="Hello")
        result = await node.run({"name": "Developer"})
        print(result)  # {'message': 'Hello, Developer!'}

    asyncio.run(main())
    ```
  </Step>
  <Step title="Learn Core Concepts">
    Understand the fundamentals:
    - [Nodes and Workflows](/flow-core/core/nodes)
    - [Chaining and Composition](/flow-core/core/chaining)
    - [Context Management](/flow-core/core/context)
  </Step>
  <Step title="Build Real Applications">
    Explore practical examples:
    - [Chat Bot with Memory](/examples/flow-core/streaming-chat)
    - [RAG Pipeline](/examples/flow-core/rag-pipeline)
    - [Parallel Processing](/examples/flow-core/parallel-search)
  </Step>
</Steps>

## Installation Options

<CodeGroup>

```bash Local Development
# Install Flow Core for Python development
pip install nadoo-flow-core

# With optional features
pip install "nadoo-flow-core[redis,cel]"
```

```yaml Docker
# docker-compose.yml
version: '3.8'
services:
  nadoo:
    image: nadooai/platform:latest
    ports:
      - "8000:8000"
    environment:
      - API_KEY=${NADOO_API_KEY}
```

```bash Cloud Deployment
# Deploy to cloud platforms
nadoo deploy --platform aws
nadoo deploy --platform gcp
nadoo deploy --platform azure
```
