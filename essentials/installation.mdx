## Verification

After installation, verify that everything is working correctly:

<CodeGroup>

```javascript Node.js
const { NadooAI } = require('@nadoo-ai/sdk');

const client = new NadooAI({
  apiKey: 'test-key'
});

console.log('Nadoo AI SDK version:', client.version);
```

```python Python
import nadoo_ai

client = nadoo_ai.Client(api_key="test-key")
print(f"Nadoo AI SDK version: {nadoo_ai.__version__}")
```

```go Go
package main

import (
    "fmt"
    nadoo "github.com/nadoo-ai/nadoo-go"
)

func main() {
    client := nadoo.NewClient("test-key")
    fmt.Printf("Nadoo AI SDK initialized: %v\n", client)
}
```

</CodeGroup>

## Docker Installation

For containerized environments:

```dockerfile
# Node.js example
FROM node:18-alpine

WORKDIR /app

COPY package*.json ./
RUN npm ci --only=production

COPY . .

ENV NADOO_API_KEY=your_api_key_here

CMD ["node", "index.js"]
```

<Accordion title="Docker Compose Example">
```yaml
version: '3.8'

services:
  app:
    build: .
    environment:
      - NADOO_API_KEY=${NADOO_API_KEY}
      - NODE_ENV=production
    ports:
      - "3000:3000"
    restart: unless-stopped
```
</Accordion>

## Cloud Deployments

### Serverless Functions

<Tabs>
  <Tab title="AWS Lambda">
    ```javascript
    // handler.js
    const { NadooAI } = require('@nadoo-ai/sdk');

    const client = new NadooAI({
      apiKey: process.env.NADOO_API_KEY
    });

    exports.handler = async (event) => {
      const response = await client.completions.create({
        model: "nadoo-gpt-4",
        prompt: event.prompt
      });

      return {
        statusCode: 200,
        body: JSON.stringify(response)
      };
    };
    ```
  </Tab>
  <Tab title="Vercel">
    ```javascript
    // api/nadoo.js
    import { NadooAI } from '@nadoo-ai/sdk';

    const client = new NadooAI({
      apiKey: process.env.NADOO_API_KEY

    }
    ```

## TypeScript Support

The SDK includes TypeScript definitions out of the box:

```typescript
import { NadooAI, CompletionResponse } from '@nadoo-ai/sdk';

const client = new NadooAI({
  apiKey: process.env.NADOO_API_KEY as string
});

async function generateText(prompt: string): Promise<CompletionResponse> {
  return await client.completions.create({
    model: "nadoo-gpt-4",
    prompt: prompt,
    maxTokens: 100,
    temperature: 0.7
  });
}
```

## Troubleshooting Installation

<AccordionGroup>
  <Accordion title="Permission Denied">
    If you encounter permission errors, try:
    ```bash
    sudo npm install -g @nadoo-ai/sdk
    # Or use a Node version manager like nvm
    ```
  </Accordion>
  <Accordion title="Network Issues">
    Behind a corporate proxy? Configure npm:
    ```bash
    npm config set proxy http://proxy.company.com:8080
    npm config set https-proxy http://proxy.company.com:8080
    ```
  </Accordion>
  <Accordion title="Version Conflicts">
    Clear your package cache:
    ```bash
    npm cache clean --force
    rm -rf node_modules package-lock.json
    npm install
    ```
  </Accordion>
  <Accordion title="Build Failures">
    Install build tools:
    ```bash
    # macOS
    xcode-select --install

    # Ubuntu/Debian
    sudo apt-get install build-essential

    # Windows
    npm install --global windows-build-tools
    ```
  </Accordion>
</AccordionGroup>

## Platform-Specific Notes

<Warning>
  Some platforms may require additional configuration.
</Warning>

### Windows

- Use PowerShell or Command Prompt as Administrator for global installations
- Consider using WSL2 for better compatibility

### macOS

- Ensure Xcode Command Line Tools are installed
- May need to allow terminal access in System Preferences > Security & Privacy

### Linux

- May require sudo for global installations
- Ensure you have proper SSL certificates installed
